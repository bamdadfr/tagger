# tagger

my own tagging tool for my audio library.

most of the things are hardcoded

## dependencies

- python3
- python3-pip
- python3-venv

## install

```bash
./install.sh
```

## configure

change `ENV_PATHS` in `src/env.py`

## run

```bash
./run.sh
```
