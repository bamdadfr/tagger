## Dependencies

- python3
- python3-pip
- python3-venv

## How to install

```bash
git clone git@github.com:bamdadsabbagh/tagger.git
cd tagger
./install.sh
```

## Configuration file

change `ENV_PATHS` in `src/env.py`

## Run

```bash
./run.sh
```
